Data Compression

Programs are growing larger and more complex each day. Hard drive memory is being quickly eaten up by videos and software. This paper explores ways to compress spoken voice recordings using the famed Linde-Buzzo-Gray (LBG) algorithm to implement a vector quantizer.

The Matlab code, located in the zip file, is an implementation of the LBG algorithm. It automatically compresses and decompresses the sound file and plays it back. You can controll the dimensionality and number of reconstruction levels used.
